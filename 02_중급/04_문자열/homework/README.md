# 과제 - 08. 문자열

## 문제 1
영문 문자열을 입력받아 **모두 대문자로** 변환해 출력하세요. (`<ctype.h>`의 `toupper` 사용)

- 파일명: `hw01.c`

## 문제 2
문자열을 입력받아 **회문(palindrome)** 인지 판정하세요. 대소문자 구분 없이.

```
입력: Level
회문 입니다.
```

- 파일명: `hw02.c`

## 문제 3
문자열에서 특정 문자 하나(`char`)의 등장 횟수를 출력하세요.

```
문장: hello world
찾을 문자: l
'l'는 3번 나타납니다.
```

- 파일명: `hw03.c`
