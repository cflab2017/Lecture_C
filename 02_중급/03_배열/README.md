# 07. 배열

## 학습 목표
- 1차원 배열의 선언, 초기화, 인덱싱.
- 배열 길이 계산 (`sizeof(arr) / sizeof(arr[0])`).
- 2차원 배열을 다룬다.

## 개념 정리

### 1) 선언과 초기화
```c
int a[5];                    // 쓰레기 값
int b[5] = {1, 2, 3};        // 나머지는 0
int c[]  = {10, 20, 30};     // 크기 자동 결정 -> 3
```

### 2) 인덱스
- 0부터 시작. 마지막 인덱스는 `길이 - 1`.
- 범위를 벗어난 접근은 **정의되지 않은 동작(UB)**.

### 3) 길이 계산 (지역 배열에서만)
```c
int arr[10];
size_t n = sizeof(arr) / sizeof(arr[0]);   // 10
```

### 4) 2차원 배열
```c
int grid[3][4];              // 3행 4열
grid[i][j] = i * 4 + j;
```
