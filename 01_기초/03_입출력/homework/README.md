# 과제 - 03. 입출력

## 문제 1
사용자에게 이름과 나이를 입력받아 다음 형식으로 출력하세요.

```
이름을 입력: 홍길동
나이를 입력: 20
홍길동님은 20세입니다.
```

- 파일명: `hw01.c`
- 입력 예: `scanf("%s", name);` (이름은 공백 없는 한 단어로 가정)

## 문제 2
세 정수 a, b, c를 입력받아 평균을 소수점 둘째 자리까지 출력하세요.

```
세 정수 입력: 10 20 33
평균: 21.00
```

- 파일명: `hw02.c`

## 문제 3
초 단위 시간(`int seconds`)을 입력받아 `시:분:초` 형식으로 출력하세요.
예: `3725` → `1:2:5`

- 파일명: `hw03.c`
