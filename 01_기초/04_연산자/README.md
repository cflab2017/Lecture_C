# 04. 연산자

## 학습 목표
- 산술/관계/논리/대입/증감/비트 연산자를 구분하고 사용한다.
- 연산자 우선순위와 결합 방향을 이해한다.
- 정수 나눗셈과 형 변환(타입 캐스팅)에 주의한다.

## 개념 정리

### 1) 산술 연산자
| 연산자 | 의미 | 예 |
|--------|------|----|
| `+` `-` `*` `/` | 사칙연산 | `7 / 2` -> `3` (정수) |
| `%` | 나머지(정수형만) | `7 % 2` -> `1` |

> **정수/정수**는 정수 결과를 줍니다. `7 / 2.0`처럼 한쪽을 실수로 만들면 `3.5`.

### 2) 증감 연산자
```c
int a = 5;
a++;     // 후위: 사용 후 증가
++a;     // 전위: 증가 후 사용
```

### 3) 관계 연산자 (결과는 0/1)
`==`, `!=`, `<`, `>`, `<=`, `>=`

### 4) 논리 연산자
`&&` (AND), `||` (OR), `!` (NOT)
**단축 평가**(short-circuit): `false && f()` 의 `f()`는 실행되지 않음.

### 5) 대입 연산자
`=`, `+=`, `-=`, `*=`, `/=`, `%=`

### 6) 비트 연산자 (참고)
`&`, `|`, `^`, `~`, `<<`, `>>`

### 7) 형 변환 (cast)
```c
int a = 7, b = 2;
double r = (double)a / b;   // 3.5
```
