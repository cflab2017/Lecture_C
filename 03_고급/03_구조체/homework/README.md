## 문제 1

`Vec2 add(Vec2 a, Vec2 b)`, `double dot(Vec2 a, Vec2 b)` 함수를 구현해
사용 예를 출력하세요.

- 파일명: `hw01.c`

각 학생의 **평균**을 출력 후, **평균이 가장 높은 학생**의 이름을 출력하세요.

- 파일명: `hw02.c`
