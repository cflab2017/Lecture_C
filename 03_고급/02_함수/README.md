# 10. 함수

## 학습 목표
- 함수의 선언/정의/호출 구조를 이해한다.
- 매개변수 전달 방식(값 전달, 포인터 전달)을 구분한다.
- 재귀 함수를 작성한다.
- `static`/`extern` 같은 저장 클래스의 의미를 가볍게 안다.

## 개념 정리

### 1) 함수의 기본
```c
/* 선언(프로토타입) */
int add(int a, int b);

/* 정의 */
int add(int a, int b) {
    return a + b;
}

/* 호출 */
int s = add(3, 4);
```

### 2) 값 전달 vs 포인터 전달
- C는 **값 전달(call by value)** 만 지원합니다.
- 함수 내부에서 호출자의 변수를 바꾸려면 **포인터를 전달**해야 합니다.

### 3) 재귀 함수
```c
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```
**종료 조건**이 반드시 필요합니다.

### 4) `static` 함수/변수
- `static` 함수: 같은 파일 내에서만 호출 가능 (외부 노출 X).
- 함수 내 `static` 지역 변수: 호출 사이 값 유지.
