# 과제 - 13. 동적 메모리

## 문제 1
사용자에게 배열 크기 N을 입력받고, N개의 정수를 입력받은 뒤
**최댓값과 최솟값**을 출력하세요. 메모리는 `malloc`/`free`로 관리.

- 파일명: `hw01.c`

## 문제 2
임의의 N×M 행렬을 동적 할당해 0으로 초기화한 뒤,
대각선만 1로 바꾸어 출력하세요. (단위행렬 비슷한 효과, N=M일 필요는 없음)

- 파일명: `hw02.c`

## 문제 3
사용자에게 문자열을 입력받아 **그 문자열의 복사본**을 동적으로 할당해 반환하는
`char *my_strdup(const char *s)` 함수를 구현하고 사용 예를 보이세요.
(`<string.h>`의 `strlen`/`memcpy` 사용 가능)

- 파일명: `hw03.c`
